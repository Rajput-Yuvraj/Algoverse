Binary Search Algorithm



Introduction

Binary Search is an efficient searching algorithm that divides a sorted array into halves to locate a target element.



Time and Space Complexity

  • Best Case: O(1)
  • Average Case: O(log2n)
  • Worst Case: O(log2n)
  • Space Complexity: O(1)
